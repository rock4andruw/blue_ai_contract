# 會議記錄整理助理

## 功能目標
- 會議語音自動辨識轉文字
- 智能摘要會議重點
- 自動提取決策事項
- 識別並整理 Action Items
- 降低人工整理時間 80% 以上

## 使用方式
```
/meeting-assistant <會議錄音檔路徑或會議逐字稿>
```

## 處理流程
1. **語音處理**（如提供錄音檔）
   - 接收音訊檔案（支援 mp3, wav, m4a 等格式）
   - 使用語音辨識 API 轉換為文字
   - 識別不同發言者

2. **內容分析**
   - 解析會議逐字稿
   - 識別會議主題和議程
   - 提取關鍵討論內容

3. **智能整理**
   - 產生會議摘要（含時間、參與者、主題）
   - 列出重點討論事項
   - 標記決策結論
   - 整理 Action Items（負責人、期限、優先級）

4. **輸出格式化**
   - 產生結構化會議記錄
   - Markdown 格式輸出
   - 可匯出為 PDF 或整合至專案管理系統

## 輸出範例
```markdown
# 會議記錄

**會議時間**: 2026-05-28 14:00-15:30
**參與人員**: [名單]
**會議主題**: Blue-AI 專案啟動會議

## 會議摘要
[一段式摘要]

## 討論重點
1. [重點一]
2. [重點二]

## 決策事項
- ✅ [決策一]
- ✅ [決策二]

## Action Items
| 任務 | 負責人 | 期限 | 優先級 | 狀態 |
|------|--------|------|--------|------|
| [任務描述] | [姓名] | [日期] | High | 待處理 |
```

## 技術整合
- 語音辨識: Azure Speech Service / Google Cloud Speech-to-Text
- AI 分析: Claude API (Sonnet 4.6+)
- 輸出格式: Markdown / JSON / PDF

## 品質保證
- 準確率目標: 語音辨識 >95%
- 重點提取準確度 >90%
- 處理速度: 1小時會議 < 5分鐘完成
