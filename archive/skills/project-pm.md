# 專案 PM 助理

## 功能目標
- 協助專案規劃與排程
- 自動追蹤專案進度與里程碑
- 風險識別與管理
- 資源分配優化
- 自動產生各類專案報告
- 團隊協作效率提升
- 降低 PM 行政工作時間 60% 以上

## 使用方式
```
/project-pm [--action <動作>] [--project <專案名稱>]
```

## 核心功能

### 1. 專案規劃 (Planning)
- **專案啟動**
  - 專案章程產生
  - 利害關係人分析
  - 目標與範圍定義
  - 成功標準設定

- **工作分解 (WBS)**
  - 自動拆解專案任務
  - 估算工作量
  - 識別相依性
  - 產生甘特圖

- **資源規劃**
  - 人力需求評估
  - 技能匹配分析
  - 預算編列
  - 時程規劃

### 2. 執行管理 (Execution)
- **任務追蹤**
  - 即時進度監控
  - 自動更新 Burndown Chart
  - 識別延遲任務
  - 提醒待辦事項

- **團隊協調**
  - 會議排程建議
  - 衝突偵測與調解
  - 溝通記錄管理
  - 決策追蹤

- **變更管理**
  - 需求變更影響分析
  - 變更申請流程
  - 版本控制
  - 變更記錄

### 3. 監控與控制 (Monitoring & Control)
- **進度監控**
  - EVM (實獲值管理)
  - 進度偏差分析
  - 預測完工日期
  - 趨勢分析

- **風險管理**
  - 風險識別
  - 風險評估 (機率 × 影響)
  - 緩解策略建議
  - 風險監控

- **品質管理**
  - 交付物檢查清單
  - 品質指標追蹤
  - 缺陷管理
  - 審查會議支援

### 4. 報告生成 (Reporting)
- **例行報告**
  - 週報 / 月報
  - 狀態儀表板
  - 里程碑報告
  - 高階主管摘要

- **臨時報告**
  - 風險報告
  - 變更影響報告
  - 資源使用報告
  - 預算執行報告

### 5. 專案收尾 (Closure)
- **交付確認**
  - 驗收清單
  - 文件完整性檢查
  - 知識移轉

- **專案回顧**
  - 經驗教訓整理
  - 績效分析
  - 最佳實踐萃取
  - 改善建議
