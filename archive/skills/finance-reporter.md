# 財務報表核對助理

## 功能目標
- 自動讀取並解析財務報表（Excel, PDF, CSV）
- 比對不同週期報表差異（週/月/季/年）
- 識別異常數據與重大變動
- 產生財務分析摘要與建議
- 提升核對效率與正確性 90% 以上

## 使用方式
```
/finance-reporter <報表檔案路徑> [--compare <比較檔案>] [--period <週期>]
```

## 處理流程
1. **報表讀取**
   - 支援多種格式: Excel (.xlsx, .xls), PDF, CSV
   - 自動識別報表類型（損益表、資產負債表、現金流量表）
   - 提取關鍵財務指標

2. **數據驗證**
   - 檢查數據完整性
   - 驗證計算邏輯（如總和、百分比）
   - 識別明顯錯誤或異常值

3. **差異分析**
   - 期間比對（本期 vs 上期）
   - 計算變動金額與變動率
   - 標記重大差異項目（如 >10% 或絕對值 >門檻）

4. **智能判斷**
   - 分析差異原因（季節性、趨勢性、突發性）
   - 識別風險指標
   - 提供初步解讀與建議

5. **報告產出**
   - 產生對比表格
   - 視覺化圖表（趨勢圖、圓餅圖）
   - 摘要報告與重點提示

## 輸出範例
```markdown
# 財務報表分析報告

**分析期間**: 2026年4月 vs 2026年3月
**報表類型**: 損益表
**產出時間**: 2026-05-28

## 整體摘要
- 總營收: $10,500,000 (↑ 8.5%)
- 營業利益: $2,100,000 (↑ 12.3%)
- 淨利: $1,580,000 (↑ 9.8%)

## 重大變動項目
### ⚠️ 需關注
1. **管銷費用** +15.2% ($850K → $980K)
   - 主要增加: 行銷費用 +$90K, 人事成本 +$40K
   - 建議: 檢視行銷 ROI

2. **應收帳款** +22.1% ($3.2M → $3.9M)
   - 可能風險: 收款週期延長
   - 建議: 加強應收管理

### ✅ 正向指標
1. **毛利率** 提升至 42.5% (+1.8%)
2. **存貨週轉率** 改善 6.3%

## 財務比率分析
| 指標 | 本期 | 上期 | 變動 | 狀態 |
|------|------|------|------|------|
| 流動比率 | 1.85 | 1.72 | +7.6% | ✅ 良好 |
| 速動比率 | 1.32 | 1.28 | +3.1% | ✅ 良好 |
| ROE | 18.5% | 17.2% | +1.3% | ✅ 改善 |

## 建議事項
1. 持續監控應收帳款，建議實施更嚴格的信用管理
2. 分析管銷費用增加的合理性與效益
3. 保持目前的毛利率水準，關注成本控管
```

## 技術整合

- 數據分析: NumPy, SciPy
- AI 分析: Claude API
- 視覺化: matplotlib, plotly

- 敏感財務數據加密處理
- 存取權限控管
- 審計軌跡記錄
