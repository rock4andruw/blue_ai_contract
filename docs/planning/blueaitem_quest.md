# Blue-AI 實作對決題目資料

附上本次 Blue-AI 實作對決的題目資料（共 8 題），請各位抽空先行預讀。

| 題號             | 問題需求                                                       | 達成目標（包含但不限於）                                                                                                                                       |
| :--------------- | :------------------------------------------------------------- | :------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **第一題** | 各式會議記錄整理耗時                                           | 會議語音辨識、重點摘要、決策事項與 Action Item 自動整理能力，降低人工整理時間，提升會議紀錄效率與準確性。                                                      |
| **第二題** | 財務報表與預算需人工核對，作業耗時                             | 自動比對每周/月/季/年財務報表差異並提出總結或基本判斷，提升核對效率與正確性。                                                                                  |
| **第三題** | 週報需人工彙整各專案進度，耗費時間                             | 串接 ADO 與 Git 收集各專案看板並自動彙整週報與最新進度或建議改善。                                                                                             |
| **第四題** | **合約文件（SLA/NDA/採購）需人工比對差異，容易遺漏異動** | **自動比對合約版本差異，快速標示異動內容，產出常用合約範本。**                                                                                           |
| **第五題** | 需求訪談後轉專案文件助理                                       | 依 PMO 提供的專案文件規範，結合客戶及標準的範本文件，直接產出需求說明書，並檢視可能的議題、優化建議、週邊系統的串接及可能風險。                                |
| **第六題** | 維運監控與異常處理建議助理                                     | 依系統 LOG 及工作說明（例如某個 JOB 要 30 分內完成），及週邊系統的回覆，提早警示是否有風險及優化建議，並且若有異常，也能第一時間通知異常點，及可能的解決方案。 |
| **第七題** | 架構圖與流程圖自動產生器助理                                   | 當使用者有需求異動時，依規範文件及系統 KH，作架構圖、需求及相關文件的調整。                                                                                    |
| **第八題** | 制式簡報與報告自動生成助理                                     | 依公司規劃或單位風格，建立如「建立金融產業專屬報告模板」，並且可以依數字或上傳表單的說明，自動產生簡報中的報表，例如營業收入或毛利說明。                       |
